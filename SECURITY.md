# Security Policy

## Reporting A Vulnerability

Please report security vulnerabilities privately through GitHub's
**Security > Report a vulnerability** feature.

Do not open a public issue for vulnerabilities that could expose users, damage
devices, or disclose sensitive information.

## Flashing Safety

OdinMac performs privileged device operations through Heimdall. Always verify
firmware against the exact Samsung device model before flashing. Never share
device serial numbers, proprietary firmware, personal logs, or private device
data in public reports.
