Welcome to the OdinMac Installer

This installs OdinMac, a native Samsung firmware flasher for Apple Silicon Macs, into your Applications folder.

This installer will:

  • Install OdinMac.app into /Applications
  • Clear the macOS quarantine flag so OdinMac opens without extra steps
  • Open OdinMac automatically once installation finishes

When OdinMac opens, it checks for optional helper tools (Homebrew, lz4, ADB) and lets you install any missing ones with a single click, right from the app. No Terminal commands are required.

Requirements: Apple Silicon Mac (M1 or later) running macOS 13 Ventura or later.
